# 使用Python執行
## 需要：
Python 3.6+

## 安裝步驟
1. 使用`pip install -r requirements.txt`安裝第三方套件
2. 安裝第三方套件後執行pixiv_downloader.py

## 其他說明
1. old_file資料夾下的檔案為以前版本的舊檔案
